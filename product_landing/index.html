About

Raspberry Pi is a small single-board computer.

It is a low cost credit-card sized computer.

Widely used in a large number of hobby projects as well as commercial ones.

Features

  • The new raspberry pi 4 has a 4 x 1.5 ghz Cortex-A72 processor
  • Fast LPDDR4 ram which comes in 3 variants 2GB, 4GB, 8GB.
  • Drives 2 displays at 4K.
  • Fast gigabit ethernet for unlimited web browsing and wireless networking for portability.
  • Energy efficient, cooler and runs silently.
